## 1.0.0 (2026-07-15)

- Public MIT release under the Visual Web Canvas name.
- Exact static HTML canvas with source-backed text, style, responsive, duplicate, delete, compare, fold, and history controls.
- Live React/Vite/Next canvas backed by React Grab and React Rewrite.
- Existing-chat Claude Code bridge with visual comments and comment-only screenshots.
- Loopback-only live proxy with Vite/Next HMR and Remote SSH WebSocket forwarding.
- Cursor MCP beta, one-command installer, diagnostics, repair, and uninstall.
- Public privacy/security documentation, examples, CI, and release packaging.
