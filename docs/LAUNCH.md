# Launch package

## Demo video — 35 to 40 seconds

1. Open a polished Next.js landing page in VS Code and connect `localhost:3000`.
2. Click the hero in Inspect mode; briefly show the resolved component and source path.
3. switch to Design, edit the headline and tighten spacing, then confirm the staged change.
4. Switch between 1440 desktop and 390 mobile.
5. Select the CTA, click Comment → Claude, and write “Make this the primary conversion action and improve contrast.”
6. Submit `do this` in the existing Claude Code chat.
7. Show Claude receiving the component, DOM, styles, comment, and screenshot; then show HMR update the canvas.
8. End on the install command and GitHub URL.

Do not show internal funnels, customer copy, browser bookmarks, personal notifications, tokens, or private file paths.
